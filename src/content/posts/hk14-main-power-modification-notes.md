## 1. 问题现象

设备在大负载工作时，`4.2V -> 16V` 电源输出不稳定，表现为负载带不动或运行异常。

已验证现象：

- 使用锂电池供电时，大负载下不稳定。
- 在 `BATT` 入口增加大电容后，仍然带不动大负载。
- 外接电源直接供 `4.2V`，仍然不能正常运行。
- 外接电源供 `8V~8.4V`，设备可以正常运行。

初步结论：

- 问题不是单纯的输出电容不足。
- 问题与输入电压和输入电流能力高度相关。
- `4.2V` 输入时，为维持 `16V` 输出，输入电流过大，导致升压级或输入路径进入极限。
- `8V~8.4V` 输入时，输入电流显著下降，所以系统可以正常工作。

## 2. 相关电源结构

原理图中相关模块位于 `Power.kicad_sch`。

`4.2V -> 16V` 升压结构：

```text
BATT
 -> Q3 PMOS 输入开关
 -> C93/C94 输入储能
 -> U6 MT3608 升压
 -> L1 6.8uH
 -> D9 SK34
 -> C28/C30 输出滤波
 -> R37 100mR 电流采样
 -> C31
 -> +16V 负载
```

关键器件：

- `U6 = MT3608`：升压芯片。
- `L1 = 6.8uH`：升压电感。
- `D9 = SK34`：肖特基整流二极管。
- `R32 = 100K`、`R33 = 3.9K`：反馈分压，设定输出约 `15.98V`。
- `R37 = 100mR, 1%, 1W`：16V 输出电流采样电阻。
- `U9 = TP181A1-CR`：电流检测放大器，输出 `16V_CUR`。
- `Q2/Q3`：`EN_16V_P` 控制的 16V 输入开关。

输出电压计算：

```text
Vout = 0.6 * (1 + 100K / 3.9K) = 15.98V
```

## 3. 电容相关改动方案

### 3.1 已讨论的 16V 输出端补电容

用途：

- 验证 `16V` 负载端是否存在瞬态储能不足。
- 改善大负载瞬间的 `16V` 掉压。
- 改善负载脉冲造成的纹波和瞬态跌落。

推荐焊接位置：

```text
大电容正极 +  -> R37 pad2
大电容负极 -  -> C31 pad2 / GND
```

已确认网络：

- `R37 pad2 = +16V`
- `C31 pad1 = +16V`
- `C31 pad2 = GND`
- `R37 pad1 = /Power/+IN`

禁止接错：

- 不要把大电容正极接到 `R37 pad1`。
- `R37 pad1` 是电流采样电阻前端，不是最终负载端 `+16V`。

等效连接：

```text
MT3608输出 ---- R37 ---- +16V ----> 负载
              pad1     pad2
                       |
                       +---- 大电容正极

GND ------------------------ 大电容负极
```

推荐电容：

- `100uF / 25V` 或 `100uF / 35V` 低 ESR 铝电解、固态电容或聚合物电容。
- 可进一步尝试 `220uF / 25V` 或 `220uF / 35V`。
- 不建议一开始直接上 `470uF~1000uF`，可能导致启动浪涌更大。

建议并联小电容：

- `10uF / 50V X7R`
- `0.1uF / 50V X7R`

判断结果：

- 如果加大电容后只改善启动瞬间，但持续大负载仍掉压，说明升压功率级不够。
- 如果脉冲负载明显改善，说明负载端储能不足是问题之一。
- 如果无明显改善，说明主要问题在输入侧或 MT3608 功率能力。

### 3.2 `R37 前` 与 `R37 后` 加电容的区别

`R37 前`：

- 节点是 MT3608 升压输出、采样电阻前。
- 主要作用是稳升压芯片输出，降低开关纹波。
- 推荐加 MLCC，例如 `10uF/50V X7R 1210` 或 `22uF/25V X7R 1210`。

`R37 后`：

- 节点是最终 `+16V` 负载端。
- 主要作用是给负载提供瞬态储能。
- 推荐加大容量低 ESR 电容。

本项目当前验证优先接 `R37 后`。

### 3.3 电容小板方案

用途：

- 比直接挂在 `C31` 小焊盘上更牢固。
- 可以把多颗电容并联，形成低阻储能模块。

小板端子：

```text
VIN+ -> R37 pad2 / +16V
VIN- -> GND / C31 pad2
```

推荐组合：

- `100uF~220uF / 25V或35V` 低 ESR 大电容 1 颗。
- `10uF / 50V X7R` 陶瓷电容 1 颗。
- `0.1uF / 50V X7R` 陶瓷电容 1 颗。

连接方式：

```text
VIN+ ---- +| 大电容
      |---|| 10uF
      |---|| 0.1uF

VIN- ---- -| 大电容
      |---|| 10uF
      |---|| 0.1uF
```

制作要点：

- 所有电容并联。
- 正极母线和负极母线尽量短、粗。
- 引线尽量短，建议控制在 `2cm~3cm` 内。
- 电容本体必须用胶固定，不能靠焊点承重。
- 上电前必须测 `VIN+` 和 `VIN-` 是否短路。

如果允许破坏 PCB：

- 可以从 `R37 pad2` 拉出短粗线作为新的 `+16V` 星形节点。
- 负极就近接地平面或 `C31 pad2`。
- 负载也尽量从这个星形节点取电。

## 4. BATT 输入端补电容方案

曾讨论在 `BATT` 电源入口增加大电容。

目的：

- 降低电池线、保护板、连接器造成的瞬态压降。
- 给 MT3608 输入侧提供短时间脉冲电流。

实际验证：

- 增加 `BATT` 入口大电容后，设备在大负载下仍然带不动。

结论：

- 单靠 `BATT` 输入端电容不能解决问题。
- 说明问题更可能是持续功率能力不足，而不是单纯瞬态滤波不足。

## 5. 电池与供电方案判断

根据验证结果：

```text
4.2V 外接电源 -> 不能正常运行
8V 外接电源   -> 可以正常运行
```

判断：

- 问题不只是锂电池本身。
- 这套电路在 `4.2V` 输入工况下，对当前负载来说余量不足。
- `4.2V -> 16V` 升压比高，输入电流大，MT3608、电感、二极管、输入路径都容易到极限。

输入电流估算：

假设 16V 输出功率为 `8W`，效率按 `85%` 估算：

```text
4.2V 输入:
Iin = 8W / (4.2V * 0.85) = 2.24A

8V 输入:
Iin = 8W / (8V * 0.85) = 1.18A
```

如果输出功率为 `16W`：

```text
4.2V 输入:
Iin = 16W / (4.2V * 0.85) = 4.48A

8V 输入:
Iin = 16W / (8V * 0.85) = 2.35A
```

结论：

- 同样负载功率下，`4.2V` 输入电流约为 `8V` 输入的 2 倍。
- 电流越大，线损、MOS 损耗、PCB 压降、电感峰值电流、芯片热损耗都会显著增加。

## 6. 长期推荐改动

### 6.1 首选：改成 2S 电池方案

推荐：

```text
2S 锂电池组
标称电压 7.4V
满电电压 8.4V
```

原因：

- 已经验证 `8V~8.4V` 输入可以正常运行。
- 输入电流明显降低。
- 对升压芯片、电感、走线、连接器压力都更小。

需要同步检查：

- 所有 `BATT` 网络上的器件耐压是否支持 `8.4V`。
- `BATT` 电容耐压是否足够。
- 电池电压 ADC 分压是否需要调整。
- 软件电量算法是否需要调整为 2S。
- 原有充电方案是否支持 2S。

注意：

- 如果原来是 1S 充电，不能直接给 2S 电池充电。
- 必须使用 2S 充电管理和 2S BMS 保护板。

### 6.2 如果必须保留 1S 锂电

需要改动：

- 更换高倍率低内阻电芯。
- 更换更大电流保护板。
- 加粗电池线。
- 降低连接器接触电阻。
- 加强 `BATT` 到升压级的铜皮和过孔。
- 重新设计 `4.2V -> 16V` 升压电源。

可能需要更换：

- 更大功率升压芯片。
- 更高饱和电流电感。
- 更低损耗整流方案，例如同步升压。
- 更适合大功率的 PCB 布局。

判断：

- 继续使用 MT3608 做大功率 `1S -> 16V`，余量可能不足。

## 7. 建议验证步骤

### 7.1 测压定位

大负载瞬间同时测以下点：

1. 外接电源输出端电压。
2. 主板 `BATT` 入口电压。
3. MT3608 输入端电压。
4. R37 前端 `/Power/+IN` 电压。
5. R37 后端 `+16V` 电压。

判断：

- 电源输出端掉压：外接电源限流或功率不足。
- 电源端不掉，BATT 掉压：线材、接口、保护板、走线问题。
- BATT 正常，16V 掉压：升压级能力不足。
- R37 前后压差过大：负载电流大，或 R37/走线损耗明显。

### 7.2 电容验证

建议按顺序验证：

1. 在 `R37 后` 加 `100uF/35V`。
2. 并联 `10uF/50V X7R` 和 `0.1uF/50V X7R`。
3. 若有改善，再试 `220uF/35V`。
4. 若仍无改善，停止继续堆电容，转向供电架构或升压级整改。

### 7.3 外接电源验证

建议设置限流并记录：

- `4.2V / 2A`
- `4.2V / 4A`
- `4.2V / 6A`
- `8.0V / 2A`
- `8.4V / 2A`

重点看：

- 是否触发电源限流。
- 输入电流峰值是多少。
- `+16V` 是否塌陷。
- MT3608、电感、D9 是否明显发热。

## 8. 焊接安全注意事项

极性电容必须确认：

- 正极接 `+16V`。
- 负极接 `GND`。
- 电解电容接反有爆裂风险。

上电前必须检查：

- `+16V` 对 `GND` 是否短路。
- `BATT` 对 `GND` 是否短路。
- 飞线是否可能碰到其他焊盘。
- 电容是否固定牢靠。

焊接机械固定：

- 大电容不能只靠焊点悬空。
- 必须用热熔胶、硅胶、AB 胶或扎带固定。

## 9. 当前建议结论

短期验证：

- 可以继续在 `R37 后` 加大电容验证负载端瞬态储能问题。
- 推荐接法：`+` 接 `R37 pad2`，`-` 接 `C31 pad2`。

中期判断：

- 如果 `4.2V` 外接电源仍带不动，而 `8V` 可以，说明不是普通锂电池问题，而是 `4.2V` 输入架构余量不足。

长期方案：

- 优先改成 `2S 7.4V/8.4V` 电池系统。
- 如果必须坚持 1S，需要重做更大功率的 `1S -> 16V` 升压电源，而不是继续单纯增加电容。
